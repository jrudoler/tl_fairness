# Experiment Reproduction

This repo reproduces the paper through generated CSV and pickle artifacts in
the repository root. The notebooks read those artifacts and render the figures.

## Setup

```bash
uv sync
```

## Generate Artifacts

Paper-size runs:

```bash
uv run python experiments/robust_experiment.py
uv run python experiments/cmi_experiment.py
uv run python experiments/adult_experiment.py --cache-importance
uv run python experiments/law_experiment.py --cache-importance
```

The generated files are:

```text
robust_res.csv
cmi_coverage.csv
cmi_compare.csv
truth_dict.pkl
adult_results.pkl
law_results.pkl
```

Small smoke-test runs:

```bash
uv run python experiments/robust_experiment.py --sample-sizes 250 --reps 2 --truth-n 100000 --output tmp_runtime/robust_probe.csv
uv run python experiments/cmi_experiment.py --weights 0 1 --sizes 500 --n-truth 200000 --sims 2 --compare-repeats 1 --coverage-output tmp_runtime/cmi_coverage_probe.csv --compare-output tmp_runtime/cmi_compare_probe.csv --truth-output tmp_runtime/truth_probe.pkl
uv run python experiments/law_experiment.py --metrics parity --importance-samples 1 --cache-importance --output tmp_runtime/law_probe.pkl
```

## Runtime Notes

Observed on this machine:

```text
Law permutation importance, 1 permutation / 10 variables: ~21.5s
Adult permutation importance, 1 permutation / 11 variables: ~42.0s
Robust probe, n=250 and 2 reps: ~0.4s for the simulation loop
CMI probe, n=500 and 2 coverage sims: ~1.0s per c value
```

Approximate paper-size runtime:

```text
Adult data analysis, original uncached importance: ~50-60 hours
Adult data analysis, --cache-importance: ~10-12 hours
Law data analysis, original uncached importance: ~30 hours
Law data analysis, --cache-importance: ~3-4 hours
Robust simulation: ~5-10 minutes
CMI simulation: likely 10+ hours, with possible memory pressure in KNN at n=10000
```

The data-analysis estimates are dominated by permutation importance. With
`n_samples=1000`, Adult asks for 11,000 prefix fits per metric and Law asks for
10,000 prefix fits per metric. `--cache-importance` reuses repeated subset
values, reducing the ceiling to 2,047 fits per Adult metric and 1,023 fits per
Law metric.
